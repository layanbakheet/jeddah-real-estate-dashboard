# Jeddah Real Estate Analytics Dashboard

This project analyzes simulated real estate data in Jeddah to identify pricing trends and key factors affecting property values.

## Project Objective

The goal of this project is to connect Data Science with Real Estate by analyzing property prices, comparing districts, visualizing market trends, and building a simple machine learning model to predict property prices.

## Features

- Interactive Streamlit dashboard
- District and property type filters
- KPI cards for total listings, average price, and average area
- Average price by district chart
- Area vs price scatter plot
- Property type distribution chart
- Jeddah district map
- Linear Regression model for price prediction
- Downloadable dataset

## Tools Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Important Note

The dataset used in this version is simulated for learning and portfolio purposes. Future versions can use real real estate listing data or official market transaction data.

## Project Summary

Analyzed real estate data to identify price trends and key factors affecting property values. Built visual dashboards to present insights and used a simple regression model to estimate property prices.
